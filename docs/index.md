## High-performance scientific computing

JAX combines NumPy-style array programming with composable transformations for
automatic differentiation, compilation, vectorisation, and parallel execution.
This workshop focuses on how those tools can be used responsibly and effectively
in scientific research.

<div class="workshop-grid">
